# Ship Detection & Intelligence API

This is the AI-powered backend for the Ship Detection project. It uses a Faster R-CNN (ResNet50-FPN) model to detect and classify ships in maritime imagery.

## Features
- **AI Detection**: Real-time ship detection using PyTorch.
- **Classification**: Rule-based intelligence for maritime zone violations.
- **FastAPI**: High-performance REST API.

## Deployment
This Space runs a Docker container exposing the FastAPI app on port 7860.
